# mariana-quinto-marques.github.io

Personal portfolio site for Mariana Quinto Marques — Senior Product Manager based in London, UK.

## Stack

Single-file static site — HTML, CSS, and JS inline. No build step.

- **Fonts:** Inter, JetBrains Mono (Google Fonts)
- **Icons:** Font Awesome 6

## Deploy

Push to `main` on a repo named `mariana-quinto-marques.github.io`. GitHub Pages serves `index.html` automatically.

## Local preview

```bash
npx serve .
```
